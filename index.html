UCI Protocol Specification

Welcome! I recently started a project creating a UCI chess engine, only to find that every source copy-pastes the same poorly-formatted document from 2006, which, without intending any offense to the original authors, was extremely clunky to read in parts, and rather unclear about much of the specification.

The intention of this site is to remedy this situation. The intent of this site is not to in any way change the standard itself, only to clarify and improve it's documentation.
