The "id" command

Subcommands

  • name

    Indicates that the rest of the command string should be interpreted as a single string containing the name of the engine.

  • author

    Indicates that the rest of the command string should be interpreted as a single string containing the name of the engine's author.

Use

The id command must be sent twice at the start of uci initialization. Once with the name subcommand, and once with the author subcommand.
